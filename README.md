# Advent of Code 2025

This repository contains my solutions for [Advent of Code 2025](https://adventofcode.com/2025) challenges, written in Python.

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/yourusername/advent-of-code-2025.git
```

2. Navigate to the right folder

```bash
cd day_x
```

3. Run the solution for a specific day

```bash
python filename.py
```
